Movie API Documentation

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

URL HTTP Method Business Logic Request body data format Response body data format
/movies GET Gets a list of movies None A JSON object holding data about all the movies
/movies/[moviename] GET Get data about a single movie by name None A JSON object holding data about a single movie containing the name, genre, description, director, image URL, and whether it's featured
